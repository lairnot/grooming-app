# App de citas para grooming

Versión MVP hecha con Flask + SQLite.

## Funciones incluidas
- Login básico
- Dashboard con métricas
- Registro de clientes
- Registro de mascotas
- Registro de servicios
- Creación y consulta de citas

## Acceso inicial
- Usuario: `admin`
- Contraseña: `groomies123`

## Cómo correrla
1. Instala dependencias:
   ```bash
   pip install -r requirements.txt
   ```
2. Ejecuta:
   ```bash
   python app.py
   ```
3. Abre en tu navegador:
   ```
   http://127.0.0.1:5000
   ```
